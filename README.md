# Forex Trading Analytics Dashboard

## Overview
This project analyzes Forex market data to evaluate trading performance using SQL and Power BI.

## Dataset
- Source: Kaggle Forex Historical Data
- Timeframe: EUR/USD 1-Hour candles
- Columns: Open, High, Low, Close, Volume

## Tools & Technologies
- SQL (MySQL)
- Power BI
- Kaggle Dataset

## Key Metrics
- Price Change
- Return %
- Volatility
- Profit & Loss (PnL)
- BUY vs SELL Distribution

## Dashboard Features
- KPI cards for performance summary
- Time-series PnL and return analysis
- Interactive date range filtering
- Clean and professional layout

## Use Case
Useful for understanding market behavior, evaluating trading strategies, and financial performance analysis.
